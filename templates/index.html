UCLA Graduate Admission Analysis

This project utilizes a robust machine learning pipeline to predict the admission rate for UCLA graduate school based on data from a Kaggle dataset. The application makes use of custom exceptions and logging to provide reliable error handling and traceability. Additionally, it demonstrates techniques in packaging and pipelining ML workflows including data preprocessing and model prediction.

Explore the interactive application to see predictions in real time.
